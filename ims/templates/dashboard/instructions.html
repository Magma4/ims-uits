Instructions

    {% if user.is_authenticated and user.is_staff and user.is_superuser %}
  1. - Please click on the avatar at the top right corner to view and edit your profile details.
  2. - Before making a request, please click on "Stock" to check for the items available and the quantity in stock then proceed.
  3. - To make an order, click on "Requisitions", select "Request Items" the key in your order and hit submit. NB. Description has to be the reason you are requesting for the items.
  4. - As an admin, you can see all orders and change the status of all orders. By default, all orders will be pending. When approved by the admin, the status of the order will change on the user's dashboard and the view requests tab in the admin "Requisitions" panel.
  5. - Please make sure you click the edit button on an order and add the item label you gave out in the description(eg.Biometric Devices 2,4,5) and update the order before giving it out.
  6. - If you are the admin that gave out the items to a user, please update the status of that particular order to released before the user leaves. Update the status to received if the items were received by you.
  7. - Admins can generate report in the "Report" panel and use a range of filters to aid in the getting the required report.
  8. - The "Employees" panel allows access to view the users using the application.
  9. - The dashboard provides an overview of products and orders. Clicking on "Order Details" beside a particular order allows you to have a preview of the order description. The pie chart provides info about all orders excluding orders with status returned. The bar chart provides info about the products available.
    10. {% else %}
  10. - Please click on the avatar at the top right corner to view and edit your profile details.

  16. - By default, all orders will be pending. When approved by the admin, the status of the order will change on the your dashboard. Or you can call any of the admins after you have made an order.
